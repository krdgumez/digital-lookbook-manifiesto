# Manifiesto Caribe – Digital Lookbook

## Descripción

Este proyecto consiste en un sitio web tipo Digital Lookbook que presenta un manifiesto visual inspirado en la estética caribeña. La propuesta combina fotografía, tipografía editorial y composición en grid para generar una experiencia visual interactiva.

## Jerarquía Visual

La jerarquía visual se construyó a partir de los siguientes principios:

- El título principal utiliza una tipografía serif (Playfair Display) para generar impacto y autoridad visual.
- El cuerpo del texto emplea una tipografía sans-serif (Inter) para mejorar la legibilidad.
- Se utilizó un sistema de grid de 6 columnas que permite traslapes estratégicos entre elementos para crear dinamismo.
- Las imágenes ocupan espacios predominantes en la composición, guiando la mirada del usuario a través del contraste de tamaños y posiciones.
- El uso de una paleta de colores basada en variables CSS refuerza la identidad visual y mantiene coherencia estética.

## Tecnologías utilizadas

- HTML5 semántico
- CSS3 (Grid Layout)
